## Installation

<Steps>
<Step>
### Install the library

<Tabs items={['bun', 'npm', 'yarn', 'pnpm']}>
<Tab value="bun">
```bash
bun add react-native-view-recorder
```
</Tab>
<Tab value="npm">
```bash
npm install react-native-view-recorder
```
</Tab>
<Tab value="yarn">
```bash
yarn add react-native-view-recorder
```
</Tab>
<Tab value="pnpm">
```bash
pnpm add react-native-view-recorder
```
</Tab>
</Tabs>
</Step>
<Step>
### Rebuild your dev client

```bash
bunx expo run:ios
bunx expo run:android
```

<Callout type="warn">
Expo Go does not include custom native modules. You need a development build (Expo Dev Client or bare React Native).
</Callout>
</Step>
</Steps>

## Peer dependencies

| Package | Version | Required |
|---------|---------|----------|
| react | >= 18 | Yes |
| react-native | >= 0.73 | Yes |
| expo | >= 54.0.0 | Optional |
