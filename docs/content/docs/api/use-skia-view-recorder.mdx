---
title: useSkiaViewRecorder
description: Hook for creating a Skia view recorder session
full: true
---

Hook that creates a Skia view recorder session. Uses `captureSkiaFrame` instead of `captureFrame` for Skia content. Returns a `viewRef` that must be passed to `SkiaRecordingView`.

## Import

```tsx
import { useSkiaViewRecorder } from "react-native-view-recorder";
```

## Signature

```tsx
function useSkiaViewRecorder(): ViewRecorder & {
  viewRef: React.RefObject<RecordingViewRef | null>;
}
```

## Returns

| Property | Type | Description |
|----------|------|-------------|
| `sessionId` | `string` | Unique session ID |
| `record` | `(options: RecordOptions) => Promise<string>` | Starts recording |
| `stop` | `() => void` | Stops an in-progress recording gracefully |
| `viewRef` | `React.RefObject` | Pass as the `viewRef` prop on `<SkiaRecordingView>` |

## Usage

```tsx
const recorder = useSkiaViewRecorder();

<SkiaRecordingView
  viewRef={recorder.viewRef}
  sessionId={recorder.sessionId}
>
  <Canvas style={{ flex: 1 }}>
    {/* Skia content */}
  </Canvas>
</SkiaRecordingView>

const path = await recorder.record({
  output: "/path/to/skia-output.mp4",
  fps: 60,
  totalFrames: 180,
  onFrame: async ({ frameIndex }) => updateAnimation(frameIndex),
});
```
